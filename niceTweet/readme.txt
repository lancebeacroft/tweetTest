Syntax:
Read all messages by ascending order of date arrived (oldest at the top!)
    nicetweet.exe

Read a user's messages and followed users
    nicetweet.exe username

Post a message
    nicetweet.exe username POSTING ""Message""
    or
    nicetweet.exe username -> ""Message""

Wall
    nicetweet.exe username WALL "" Text"" 

Follow
    nicetweet.exe username FOLLOWING FollowedUserName

Help
    nicetweet.exe ?
